Welcome to VoidForge

Credential Vault

Your API keys are encrypted with a password you choose. The vault is stored at ~/.voidforge/vault.enc — AES-256-GCM encrypted. You manage the password however you like (memory, 1Password, keychain, etc.).

This password encrypts your credentials locally. It never leaves your machine.

Anthropic API Key

Required for PRD generation with Claude.

Get your key at console.anthropic.com

Simple Setup Quick Start

Get straight to building. Your project runs locally or with Docker. You can deploy anywhere later — just add credentials when you're ready.

  • Project scaffolding with all VoidForge methodology
  • PRD generation with Claude
  • Docker deploy target available out of the box
Advanced Setup Cloud deploy

Connect cloud providers now so the deploy wizard (Haku) can provision infrastructure later. Best if you already have accounts with AWS, Vercel, Railway, or Cloudflare.

  • Everything in Simple, plus:
  • Connect AWS, Vercel, Railway, or Cloudflare credentials
  • Deploy target selection filtered by your available providers
  • Credentials stored for Haku to use after /build
